# FEM4C Docs Index

FEM4C のドキュメントは、このページを入口に参照してください。

## Canonical Documents
- `tutorial_manual.md`
  学習者向けの主教材。理論->実装->検証の順で進める。
- `implementation_guide.md`
  実装者向けのモジュール解説と最小検算手順。
- `01_requirements.md`
  現在の機能要件と制約（2D/線形静解析前提）。
- `03_design.md`
  モジュール分割とCLI/入出力設計の要点。
- `00_tutorial_requirements.md`
  チュートリアル設計の到達目標と学習スコープ。
- `FEM_LEARNING_GUIDE.md`
  理論寄りの補助資料（要素/行列の背景説明）。

## Operation Rules
- 新規資料を作る前に、上記の既存ファイルへ追記できないか確認する。
- 一時検討メモや作業ログは `docs/` 直下に増やさない。
- 役目を終えた資料は削除し、必要なら Git 履歴を参照する。
