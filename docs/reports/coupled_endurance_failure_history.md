# Coupled Endurance Failure History (Archived)

This page previously tracked the Slack/notification artifacts emitted by
`tools/report_archive_failure_rate.py`. Project Chrono の移植と教育資料へ集中する
方針に従い、`data/endurance_archive/` 以下のグラフや Slack 用 JSON は生成して
いません。本ファイルは過去の作業ログとしてのみ残しています。

- `data/coupled_constraint_endurance.csv` が唯一の公式ログです。
- `tools/report_archive_failure_rate.py` や関連スクリプトはリポジトリから削除済みです。

必要に応じて個人環境で再実行する場合は、`coupled_endurance.yml` から得た Run ID
やアーティファクトを手元で解析し、成果を別チャネルで共有してください。本ドキュメントに
追記する運用は終了しました。
